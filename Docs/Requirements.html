  Requirements

To install and use FilWip, your system needs to satisfy the following basic requirements:

  • Intel or compatible, PPC is not yet supported*
  • BeOS R5 or above
  • 32 MB RAM
  • 2 MB of free space in BeOS

* The PPC build of FilWip has been built but on an x86 machine, so testing was not possible. Also many-a-time this cross-platform build seems to fail on an actual PPC. Anyway, if you are interested in a PPC version and are willing to compile and test for the PPC platform, you can contact me regarding any problems that you may encounter with the code compiling under PPC.

If you have a full working version for the PPC, you can mail me the source (zipped) so I can put it up on the web for other PPC users. If the existing source works under PPC without any modifications you need not send me the source, just inform me that it works.

For compiling the source code of FilWip, you can use the BeIDE. If you are not fond of .proj files, you can probably write a makefile and compile. Before jumping into the source code, I suggest you read the License agreement for using the code.
