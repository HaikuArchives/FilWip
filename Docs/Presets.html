Presets

A preset is simply all your clean-up options saved in a file. Thus a preset is a file that contains key/value pairs regarding what options are selected. When you double-click such a file from Tracker, FilWip will load all the settings from that file to the main window, ready for you to click the Clean up button.

This is the primary and probably the only use of presets in FilWip. To load a particular set of options for quick loading and cleaning, saving you the time of re-selecting the options each time.

You can create several presets for different needs, like a preset called "Net options" that will have all internet based cache options selected.

To save a preset, open FilWip and choose the options you want to save. Then click the Save toolbar button. By default you will be prompted to save the preset into the Presets folder in ~/config/settings/FilWip/, but you can save it anywhere. The advantage of saving the preset in the Presets folder is that when FilWip loads, all presets in the folder will be added to the Preset menu (like shown in the picture), so you can quickly open it without having to navigate to the folder where you saved it in.

The "Default" preset shown is actually FilWip's settings file displayed under that name. If you load other presets and you want to revert back to the old settings file (before you close FilWip) you can choose the Default preset. You can have other presets with the name Default, but the settings file preset will always be the first in the list so you can distinguish it from other presets named Default (if there are any).

Presets can also be loaded from the command-line, see topic "The Terminal side".
