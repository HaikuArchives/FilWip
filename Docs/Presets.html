  Presets

A preset is simply all your clean-up options saved in a file. Thus a preset is a file that contains key/value pairs regarding what options are selected. When you double-click this file from Tracker, FilWip will load all the settings from that file to the main window, ready for you to click the Clean-Up! button. Now all you have to do is click the button.

This is the primary and probably the only use of presets in FilWip. To load a particular set of options for quick loading and cleaning, saving you the time of re-selecting the options each time.

You can create several presets for different needs, like say a preset called "Net Options" that will have all Internet based cache options selected.

To save a preset, open FilWip, choose the options you want to save. Then click the Save toolbar button. By default you will be prompted to save the preset into the Presets folder in FilWip, but you can save it anywhere. The advantage of saving the preset in the Presets folder is that when FilWip loads, all presets in the folder will be added to the Preset menu (like shown in the picture) so you can quickly open it without having to navigate to the folder where you saved it in.

The reason why I introduced Presets was I needed two sets of settings. One to clean Internet stuff alone (leaving behind Trash, history etc.) and another one cleaning all the options. Because setting and resetting the options each time was time consuming, I wrote Presets and ever since life is much easier :)

The "Default" preset shown is actually FilWip's settings file displayed under that name. If you load other presets and you want to revert back to the old settings file (before you close FilWip) you can choose the Default preset. You can have other presets with the name Default, but the settings file preset will always be the first in the list so you can distinguish it from other presets named Default (if any).
