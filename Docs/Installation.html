  Intallation

FilWip is distributed as a Zip file and within this archive is a SoftwareValet Install program. To un-archive the Zip file, double click on it and this will usually bring up the Expand-O-Matic or any other program capable of extracting Zip files. Extract the Zip file into a temporary destination folder.

Once the package has been un-archived to a destination folder, locate the package file and double click it. Simply follow the instructions on screen and complete the installation. The install program provides you an option of whether source code is to be installed along with the program or not. Also before you install you must agree to the License agreement of FilWip.

Once the installation is fully complete you can delete the temporary destination folder. But it is recommended you keep the original Zip archive intact (atleast till the next release), in case you wish to re-install FilWip.

If you wish to move FilWip to another location, you can do so even after the installation since all of its data files are self-contained in the installed folder. You can keep FilWip folder anywhere as long as you keep the sub-folders in the same hierarchy.
