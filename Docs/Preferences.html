Preferences

FilWip splits the set of options into three categories:


At start-up

Live monitoring
This means all files/folders targetted by the plugins will be monitored for any change.
For example, when you have FilWip open and some background app creates/deletes some temporary files they will show up in FilWip live - as and when they change.

Select as needed
If a plugin has anything to clean, it is selected, otherwise it is deselected. This will happen only once at startup.


Before clean up

Ask confirmation
You'll be asked for confirmation after choosing to clean up.


After clean up

Show report
Will show a report detailing the number of files deleted, the gained disk space etc.

Deselect all
All plugin-items will be deselected (and if you close FilWip it will save all items as deselected).

Quit
FilWip will automatically quit after the clean up action has finished.
