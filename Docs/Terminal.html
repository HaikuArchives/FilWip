The Terminal Side

FilWip offers a neat way to make it act like a simple script program. It can come in handy, not having to open FilWip everytime you want to do a cleaning-up. A simple "silent" mode allows you to do this. Command-line parameters are explained below.

Parameter Description
-preset <preset_file> Loads the items from the specified the Preset file and launches the graphical user interface (if necessary)
-silent Begins the clean-up process in silent mode. In silent mode, the graphical user interface is not launched, graphical user pop-ups/errors or report are not shown. Thery are shown in text mode instead.

Please note that both the above options can be combined, but only in the order specified i.e. -preset first, -silent next. This means you can load any Preset file and clean-up in silent mode.

Specifying only a Preset file (without the -silent option) at the command-line will launch the graphical user interface and the specified Preset (if it exists) will be loaded and ready. Note that you must specify the full path (or relative path) to the Preset file. FilWip will not check the list of Presets in the Presets folder for you.
